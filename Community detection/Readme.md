Experiment file usage Instruction

This repository contains the script **Experiment1.py**, which runs the main experiment for our project.  
Follow the steps below to correctly set up your environment and execute the experiment.

**Python** 3.9 or later (3.10+ recommended)  
**pip**  
**set TF_ENABLE_ONEDNN_OPTS=0** to avoid oneDNN errors
**set OPENAI_API_KEY**
